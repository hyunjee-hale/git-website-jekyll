This site is a collection of practical notes on Git, the command line, and developer tools — the kind of reference I wish I'd had when I was first learning.

## What you'll find here

- **Git workflows** — branching strategies, rebasing, resolving conflicts, and undoing mistakes
- **Command-line tips** — shortcuts and patterns that make everyday work faster
- **Tool deep-dives** — Jekyll, GitHub Actions, and other utilities worth knowing *(coming soon)*
